# 事件绑定指令

v-on:事件名称="事件处理函数名"

缩写@事件名称="事件处理函数名

用于监听DOM事件

<button v-on:click="add">点击+1</button>

methods: { //指定事件处理方法, 在模板页面中通过 v-on:事件名 来调用
    add: function () { //key为方法名
        console.log('add被调用')
    }
}

# 事件修饰符

@click.stop="doThis"

.stop 阻止单击事件继续传播 event.stopPropagation()

.prevent 阻止事件默认行为 event.preventDefault()

.once 点击事件将只会触发一次

# 按键修饰符

格式： v-on:keyup.按键名 或 @keyup.按键名

常用按键名：

.enter

.tab

.delete (捕获“删除”和“退格”键)

.esc

.space

.up

.down

.left

.right

<input @keyup.enter="keyEnter"> 进入输入框按回车时调用keyEnter
